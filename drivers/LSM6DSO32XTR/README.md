# LSM6DSO32XTR

This code is using the LSM6DSO32XTR chip to read accelerometer and gyroscope data.
The LSM6DSO32XTR is a 6-axis inertial measurement unit (IMU) that includes a 3-axis accelerometer and a 3-axis gyroscope.

[LSM6DSO32XTR Datasheet](https://www.st.com/resource/en/datasheet/lsm6dso32x.pdf)
